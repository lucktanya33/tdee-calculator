# TDEE計算機
原生 JS 撰寫的 TDEE 計算機，為健身減重者而設計！輸入身高體重可以算出基礎代謝和每日消耗熱量，根據你的增肌減脂目標(公斤數/時間)，可以得到每天應該攝取的熱量  
↳ [連結](https://lucktanya33.github.io/tdee-calculator/index.html)
